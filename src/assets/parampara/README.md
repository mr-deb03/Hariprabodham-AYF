# Parampara guru portraits

Drop the six guru portrait images into this folder using **exactly** these
filenames. Once they are here, the Parampara page will use them automatically
(the imports are already prepared in `src/components/ParamparaLineage.jsx`).

| # | Filename                    | Guru                                                      |
|---|-----------------------------|----------------------------------------------------------|
| 1 | `swaminarayan-bhagwan.jpg`  | Bhagwan Shree Swaminarayan                                |
| 2 | `gunatitanand-swami.jpg`    | Mool Aksharmurti Gunatitanand Swamiji                     |
| 3 | `shastriji-maharaj.jpg`     | Brahmaswaroop Shastriji Maharaj                           |
| 4 | `yogiji-maharaj.jpg`        | Brahmaswaroop Yogiji Maharaj                              |
| 5 | `hariprasad-swamiji.jpg`    | H.D.H. Hariprasad Swamiji Maharaj                         |
| 6 | `prabodh-swamiji.jpg`       | H.D.H. Pragat Guruhari Prabodh Swamiji Maharaj           |

Recommended: portrait orientation, roughly 4:5 ratio, at least 800px wide,
`.jpg` or `.webp`. If you use a different extension, update the imports in
`ParamparaLineage.jsx` to match.

> Use only images you have the rights to publish.
